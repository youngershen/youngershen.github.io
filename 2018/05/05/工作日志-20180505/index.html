工作日志-20180505
2018-05-05|工作

摘要

  • 今天上午把 「酒店大数据平台」客户的架构图出了，看了客户给的竞品之后才发现原来客户要做的是一个 2c 的乙方系统，我一直一位客户想要一个 API 而已，看来还是得多和客户沟通，不然要走弯路。

  • 今天上午同时把 「虚拟笔交易平台」客户的流程图和交易系统的建模做了出来，客户说明天给反馈。

  • 而且今天把 「保单爬虫」客户的合同重新做了一下，打算周一给客户邮寄合同，然后就马上开始开发。

备注

「酒店大数据」这个项目看来还是有点机会的，「区块链交易系统」的客户是技术的，我作为他的手下来搞这个系统，那么我所有的设计都需要他来确认，所以我觉得这个项目 2 个月能做出来就算是万幸，搞不好要做到过年，所以可以适当把时间花在别的地方。

今天注册了一个微信公众号，叫做 「银河系朋克编程指南」打算把自己这几年来独立开发的经验写一写，另外也想写一些教程类的东西，希望能够坚持下来。
